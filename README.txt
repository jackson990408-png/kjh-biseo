▌ KJH비서란?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
이 컴퓨터 전용으로 만든 개인 AI 비서입니다.
대화 내용이 인터넷으로 나가지 않으며, 설치 후에는 무료입니다.

할 수 있는 일:
  ✔ 컴퓨터 파일 검색 — "○○ 파일 찾아줘"
  ✔ 엑셀·워드·PDF·PPT 문서 분석 및 직접 생성
  ✔ CAD 도면(DXF) 분석, 스케치업 Ruby 스크립트 작성
  ✔ 인터넷 검색 (뉴스·위키) 및 링크 요약
  ✔ 프로그램 실행 — "캐드 열어줘", "크롬 켜줘"
  ✔ 일정 등록·수정·알림 (개인/회사 구분)
  ✔ 음성 입력(마이크) · 음성 답변
  ✔ 이미지 붙여넣기 분석, 화면 캡처 분석
  ✔ 민감 정보 암호화 금고 (/금고)
  ✔ 업데이트 자동 알림 (인터넷 연결 시)

▌ 처음 설치하기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 이 ZIP 파일의 압축을 해제합니다
   예) 바탕화면이나 C:\ 아무 곳에나

2. 압축 해제된 폴더 안의
   【설치시작.bat】 더블클릭

3. "이 앱이 변경을 허용하겠습니까?" → 【예】 클릭

4. 완료까지 기다리기 (10~40분, AI 모델 다운로드 포함)

5. 완료 후 바탕화면의 【KJH비서】 아이콘으로 실행
   단축키: Ctrl + Alt + K

권장 사양:
  OS     : Windows 10 / 11 (64비트)
  RAM    : 16GB 이상 권장
  그래픽  : NVIDIA GPU VRAM 6GB 이상 권장 (없어도 동작)
  저장공간: 여유 15GB 이상 (AI 모델)
  인터넷  : 설치·업데이트 시에만 필요 / 대화는 오프라인

▌ 업데이트하기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
방법 A — 자동 알림 (권장)
  앱을 실행하고 인터넷이 연결되어 있으면,
  새 버전이 나왔을 때 앱 상단에 노란색 알림이 뜹니다.
  【지금 업데이트】 버튼을 누르면 자동으로 다운받고 재시작합니다.

방법 B — 수동 업데이트
  배포 폴더의 【KJH비서_업데이트.bat】를 실행하거나
  설치된 폴더(%USERPROFILE%\AI비서\업데이트.bat) 를 실행

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
▌ 제거하기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  【KJH비서_제거.bat】를 실행합니다.
  개인 데이터(대화기록, 금고, 프로필) 보존 여부를 선택할 수 있습니다.

▌ 자주 쓰는 채팅 명령어
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  /말투 간결    — 답변을 짧게
  /말투 자세히  — 답변을 길고 자세하게
  /일정         — 일정(달력) 창 열기
  /메모         — 메모 창 열기
  /일정요약     — 전체 일정 요약
  /일정 개인    — 개인 일정만 보기
  /일정 회사    — 회사 일정만 보기
  /이름         — 비서 이름 변경
  /찾기 키워드  — 예전 대화 검색
  /금고 목록    — 저장된 비밀번호 목록
